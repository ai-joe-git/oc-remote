# OC Remote v1.6.1 — Release Notes

Maintenance release focused on shell-input ergonomics, visual polish, and documentation accuracy.

## Highlights

- Improved chat composer shell flow: long-press on Send now toggles shell mode consistently, including when the shell toggle button is hidden.
- Refined shell UX feedback in chat: clearer active-shell hint, cleaned send feedback (no success toast spam), and updated AMOLED send-button styling for better contrast.
- Tightened composer spacing and control sizing for a cleaner layout around model/agent row and input area.
- Updated README claims to match current app behavior and corrected connection/setup command examples.
- Refreshed locale files through lokit after English string updates.
